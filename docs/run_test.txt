###################### GENERAL TEST INFORMATION ######################

The program is tested using 'unittest' framework implementation in python.

The sections below gives you the commands for running test on a single module
or running all tests once.


###################### RUN TEST API ######################

python -m test.api.test_api_request_adapter
python -m test.api.test_url_querystring_builder

###################### RUN TEST APP ######################

python -m test.app.test_session

###################### RUN TEST BOT ######################

python -m test.bot.test_bot

###################### RUN TEST DATABASE ######################

python -m test.database.test_db_conn_adapter
python -m test.database.test_sql_query_builder
python -m test.database.test_db_settings_builder

###################### RUN TEST PARSER ######################

python -m test.parser.test_file_parser
python -m test.parser.test_db_answer_parser

###################### RUN TEST RUNNER ######################

python -m test.runner.test_runner




###################### RUN ALL TESTS ######################

python -m unittest discover -s test -p 'test_*.py'
