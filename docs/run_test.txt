###################### GENERAL TEST INFORMATION ######################

The program is tested using 'unittest' framework implementation in python.

The sections below gives you the commands for running test on a single module
or running all tests once.


###################### RUN TEST RESOURCE LOADER ######################

python -m test.app.test_resource_loader

###################### RUN TEST SESSION ######################

python -m test.app.test_session

###################### RUN TEST RUNNER ######################

python -m test.runner.test_runner

###################### RUN TEST CONNECTION ######################

python -m test.conn.test_conn

###################### RUN TEST SQL QUERY BUILDER ######################

python -m test.conn.test_builder

###################### RUN TEST PARSER ######################

python -m test.parser.test_parser

###################### RUN TEST BOT ######################

python -m test.bot.test_bot

###################### RUN TEST API REQUEST ######################

python -m test.api.test_api

###################### RUN URL QUERYSTRING BUILDER ######################

python -m test.api.test_builder

###################### RUN TEST DATABASE RESPONSE PARSER ######################

python -m test.parser.test_db_resp_parser

###################### RUN TEST DATABASE SETTINGS BUILDER ######################

python -m test.app.test_db_settings_builder


###################### RUN ALL TESTS ######################

python -m unittest discover -s test -p 'test_*.py'
